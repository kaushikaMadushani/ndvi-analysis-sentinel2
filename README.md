# NDVI Analysis using Sentinel-2

## Description
This project demonstrates a basic NDVI calculation using Sentinel-2 satellite imagery.

## Data
- Sentinel-2 imagery (ESA)

## Tools
- Python
- rasterio
- numpy

## Method
1. Load satellite bands
2. Calculate NDVI
3. Visualize the result

## Output
- NDVI map of the study area
